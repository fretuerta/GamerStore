Instalación del paquete GamerStore.war

Crear base de datos en MariaDB con el nombre GamerStore.
Usuario: root
Password: root

Copiar el fichero GamerStore.war en el directorio 'webapps' de Apache Tomcat.

Iniciando Apache Tomcat, automáticamente se desplegará y se servirá.

Para acceder a la aplicación:
http://localhost:8080/GamerStore/
